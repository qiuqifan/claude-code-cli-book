## 📖 关于本项目

Claude Code 是 Anthropic 官方推出的 AI 辅助编程 CLI 工具。本项目通过深度源码分析，系统性地解析其架构设计、技术选型和实现细节。

**适合人群：**
- 🎯 想要理解 AI Agent CLI 工具架构的开发者
- 🎯 研究大型 TypeScript 项目设计的工程师
- 🎯 对终端 UI、状态管理、分布式任务等技术感兴趣的读者
