# Claude Code CLI 技术手册 - 导航指南

欢迎来到 Claude Code CLI 技术手册。本手册提供多种导航方式，帮助你从不同角度理解系统架构。

## 📚 导航方式

### 1. 按架构层次导航 (推荐入门)
[`by-layer.md`](./by-layer.md) - 从底层到上层，系统化学习整体架构

适合：想要全面理解系统架构的读者

### 2. 按功能模块导航
[`by-feature.md`](./by-feature.md) - 按照功能领域分组学习

适合：想要深入了解特定功能领域的读者

### 3. 按难度级别导航
[`by-difficulty.md`](./by-difficulty.md) - 从简单到复杂，循序渐进学习

适合：初学者和希望逐步深入的读者

### 4. 依赖关系图
[`dependency-graph.md`](./dependency-graph.md) - 可视化模块间的依赖关系

适合：想要理解模块间交互和依赖的读者

## 🎯 快速开始

### 如果你是初学者
建议路径：[按难度级别导航](./by-difficulty.md) → [按架构层次导航](./by-layer.md)

### 如果你有经验
建议路径：[按功能模块导航](./by-feature.md) + [依赖关系图](./dependency-graph.md)

### 如果你要解决特定问题
使用 [按功能模块导航](./by-feature.md) 快速定位相关章节

## 📖 学习建议

1. **先读总览**: 从 [00-overview](../chapters/00-overview.md) 开始，建立整体认知
2. **选择路径**: 根据你的背景选择合适的导航方式
3. **动手实践**: 每个章节都包含实际代码示例
4. **参考索引**: 使用 [术语表](../glossary.md) 查询专业术语
